## 搬瓦工续费的核心机制：先搞懂这两件事

**搬瓦工不会主动扣你的钱。** 这是第一件要理解的事。

不管你当初是用支付宝还是PayPal买的，到期之后搬瓦工都不会自动从里面划款。它只会在到期当天给你的注册邮箱发一封标题为"Customer Invoice"的续费账单邮件。

收到邮件之后，你有 **7天时间** 去支付这张账单。7天内完成付款，VPS正常续用；超过7天没动静，VPS会被暂停；再拖几天（大约到期后第9天前后），数据就直接删了。

第二件事：**续费价格锁定在你最初购买时的价格。** 搬瓦工的套餐这些年涨价不少，但已经买入的用户，只要一直按时续费，就一直按当年的价格来。当时用了优惠码的，续费时也会自动沿用那个优惠码带来的折扣价，不用重新输入。

## 能不能提前续费？变通方法：提前充值账户余额

不能。这是最常被问到的问题，也是让人最头疼的限制。

搬瓦工官方明确不支持提前续费——账单没出来之前，根本没有"续费"这个入口可以点。

但有个变通方法：**提前充值账户余额（Add Funds）。**

搬瓦工有一个账户内置余额系统，类似预付款。你可以提前把下一期续费金额充进去。等到期当天，系统生成账单之后，如果账户余额足够，会自动扣款完成续费，全程不需要你手动操作。

举个例子：你的套餐是 $49.99/年，担心到时候忘了续费，就提前往账户充 $50 进去。到期那天账单一出，自动搞定，省心。

甚至有人图省事，一次性充好几年的金额进去。技术上完全可以，相当于"续费好几年"。搬瓦工的服务稳定性有口皆碑，敢这么操作的用户不少。

充值方式同样支持支付宝，操作路径：登录后台 → 点击账号名 → 选"Add Funds" → 输入金额 → 选择支付宝扫码。

## 搬瓦工后台设置：确认"Renewal Invoices"是打开状态

这件事很容易被忽略，但其实挺关键。

搬瓦工后台里每个VPS服务旁边都有一个"Renewal Invoices"开关，默认是 **ON（开启）** 状态，代表到期时会生成续费账单并发邮件提醒。

如果这个开关被手动关成了 **OFF**，到期时系统不会生成任何账单，VPS也不会有续费机会，直接到期删除。

建议每隔一段时间登录后台检查一眼：进入"My Services"→ 找到对应VPS → 点击管理 → 查看"Manage renewal"选项，确认是ON状态。

## 续费遇到问题？几个常见情况处理

**问题一：续费后系统没反应，VPS状态没变**

付款之后如果发现VPS状态没有立即更新，先等几个小时。搬瓦工的后台有时候处理稍慢，不是马上刷新。如果等了一天还是没变化，登录后台提交工单，附上支付宝或PayPal的交易号，客服会手动核对处理。

**问题二：邮件收不到，但VPS已经到期**

直接登录搬瓦工后台（`bwh81.net`），进入"My Invoices"查看所有账单状态，找到未付款的续费账单直接支付即可，不依赖邮件也能完成续费。

**问题三：VPS已经被暂停，还能续费吗**

可以。暂停状态（Suspended）不是删除，账单还在。只要还在7天的宽限期内，支付账单后VPS会恢复正常。超过这个窗口期，数据就没了。

## 搬瓦工续费常见问题（FAQ）

**Q：搬瓦工续费价格会比购买时贵吗？**

A：不会。续费价格锁定在首次购买时的价格，包括当时使用的优惠码折扣。即使官方后来上调了套餐售价，你的续费价格也不受影响。唯一例外是官方统一调整了某款套餐的定价，这种情况历史上也有发生过，但属于例外而非常态。

**Q：续费可以用支付宝吗？**

A：可以。搬瓦工目前支持支付宝、银联、PayPal 和信用卡四种支付方式，国内用户用支付宝扫码支付非常方便。账户余额充值同样支持支付宝。

**Q：超过7天没续费，数据还在吗？**

A：到期后7天左右VPS会被暂停，约第9天前后数据会被删除。暂停期间仍可续费恢复，但一旦删除就无法找回了。所以续费邮件来了要尽快处理，或者提前充值余额自动续费更安全。

**Q：想续费两年或三年可以吗？**

A：搬瓦工不支持直接购买多年套餐，但你可以向账户余额充入多年金额（比如年付套餐充2倍价格），系统每次到期自动从余额扣款，实际效果等同于续费多年。

**Q：不想继续用了，需要主动取消吗？**

A：不需要。直接忽略续费账单不支付即可，搬瓦工不会自动扣款，也不会有额外处罚。如果不想收续费提醒邮件，可以在后台把对应VPS的"Renewal Invoices"开关关到OFF。
